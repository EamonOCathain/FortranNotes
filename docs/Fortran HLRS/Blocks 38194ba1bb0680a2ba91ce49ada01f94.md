# Blocks

## Block

BLOCK creates a local scope. Variables declared inside the block
exist only between BLOCK and END BLOCK.

Example:

```fortran
integer :: i
i = 3

block
    integer :: i
    i = 10
    print *, i
end block

print *, i
```

Output:

```fortran
10
3
```

The inner variable exists only inside the block and does not affect
the outer variable.

Named blocks are allowed:

```fortran
search : block
    ...
    if (found) exit search
    ...
end block search
```

Useful for:

- Temporary variables
- Limiting variable scope
- Exiting a section of code with EXIT

Less common than IF, DO, and SELECT CASE, but occasionally seen in
modern Fortran.
