# Arrays

## **Arrays**

### **Static Arrays**

Size known when declared.

```fortran
integer, dimension(24) :: arr
```

### **Allocatable Arrays**

Size determined during program execution.

```fortran
integer, allocatable :: arr(:)
```

Allocate memory later:

```fortran
allocate(arr(100))
```

Fortran arrays are 1-indexed:

```fortran
arr(1) = 10
arr(2) = 20
```

### **Multi-Dimensional Arrays (Matrices)**

Arrays can have multiple dimensions:

```fortran
real :: matrix(3,4)
```

or

```fortran
real, dimension(3,4) :: matrix
```

This creates a **3 × 4** array.

Access elements using:

```fortran
matrix(2,3)
```

---

### **Array Size**

The size of an array can be queried using:

```fortran
size(arr)
```

Example:

```fortran
integer :: arr(24)

print *, size(arr)
```

Output:

```
24
```

---

# **Array Indexing and Slices**

Accessing a single element:

```fortran
arr(1)
```

returns the first element of a 1D array.

A subset of an array can be selected using `:`.

```fortran
arr(1:5)
```

returns elements 1 through 5.

```fortran
arr(:)
```

returns the entire array.

---

### **Multiple Dimensions**

For multidimensional arrays, one index is provided for each dimension.

```fortran
arr(i,j)
```

returns the element at row `i`, column `j`.

```fortran
arr(i,j,k)
```

returns the element at position `(i,j,k)` in a 3D array.

Examples:

```fortran
arr(2,3)
```

element in row 2, column 3.

```fortran
arr(10,20,5)
```

element with indices `(10,20,5)`.

---

### **Array Sections**

The `:` operator can be used on individual dimensions.

```fortran
arr(2,:)
```

returns all columns from row 2.

```fortran
arr(:,3)
```

returns all rows from column 3.

```fortran
arr(:,:)
```

returns the entire 2D array.

```fortran
arr(:,:,1)
```

returns the first slice of a 3D array.

---

### **Summary**

| **Syntax** | **Meaning** |
| --- | --- |
| `arr(i)` | One element of a 1D array |
| `arr(i,j)` | One element of a 2D array |
| `arr(i,j,k)` | One element of a 3D array |
| `arr(:)` | Entire 1D array |
| `arr(i,:)` | Row `i` |
| `arr(:,j)` | Column `j` |
| `arr(:,:)` | Entire 2D array |
| `arr(:,:,k)` | Slice `k` of a 3D array |

This is probably enough for now. Later, when you reach scientific-model code, you’ll start seeing things like `arr(:, :, t)` and `arr(i, :, :)` constantly, especially for climate model state variables stored as `(lat, lon, time)` or similar.

---

# **Whole-Array Operations**

Fortran can operate on entire arrays without explicit loops.

```fortran
real :: a(3), b(3)

a = [1.0, 2.0, 3.0]
b = a + 10.0
```

Result:

```
b = [11.0, 12.0, 13.0]
```

Operations are applied element-wise:

```fortran
c = a + b
d = a * b
a = 0.0
```

No loop is required.

---

### **Key Ideas**

- Arrays can have one or more dimensions.
- `size()` returns the number of elements.
- `:` is used for array slicing.
- Array arithmetic is performed element-wise.
- Whole-array operations are one of Fortran’s most powerful features.
