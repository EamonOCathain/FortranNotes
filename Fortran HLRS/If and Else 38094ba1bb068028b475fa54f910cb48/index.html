    If and Else¶

    Summary¶

    Statement Purpose
    if (...) then Execute code if condition is true
    else Alternative branch
    else if (...) then Test another condition
    == Equal
    /= Not equal
    .and. Logical AND
    .or. Logical OR
    .not. Logical NOT

    This covers essentially all conditional logic used in everyday Fortran programs.

    Basic IF Statement¶

    Execute code only if a condition is true.

    if (x > 0) then
    print *, "Positive"
end if

    IF … ELSE¶

    Choose between two alternatives.

    if (x > 0) then
    print *, "Positive"
else
    print *, "Zero or Negative"
end if

    IF … ELSE IF … ELSE¶

    Test multiple conditions in order.

    if (x > 0) then
    print *, "Positive"
else if (x < 0) then
    print *, "Negative"
else
    print *, "Zero"
end if

    Common Example: Divisibility Test¶

    Using modulus to check if division has a remainder

    if (mod(n,2) == 0) then
    print *, "Even"
else
    print *, "Odd"
end if

    Single-Line IF¶

    For a single statement, IF can be written on one line:

    if (x /= y) call answer(x, y)

    Equivalent to:

    if (x /= y) then
    call answer(x, y)
end if

    Named IF Constructs¶

    Long IF blocks can be given a name:

    traffic_light : if (color == 'red') then
    ...
end if traffic_light

    Useful for large nested code blocks.

    ELSE IF Behaviour¶

    Once a condition is true and its branch executes, the remaining ELSE IF conditions are not evaluated.

    Logical Expression Warning¶

    Fortran does not guarantee the order of evaluation of complex logical expressions.

    Avoid code like:

    if (i > 0 .and. a(i) > 5) then

    because a(i) may be evaluated even when i <= 0.

    Safer:

    if (i > 0) then
    if (a(i) > 5) then
        ...
    end if
end if

    Comparison Operators¶

    Operator Meaning
    == Equal to
    /= Not equal to
    < Less than
    <= Less than or equal to
    > Greater than
    >= Greater than or equal to

    Example:

    if (a /= b) then
    print *, "Different"
end if

    Logical Operators¶

    Operator Meaning
    .and. AND
    .or. OR
    .not. NOT

    Example:

    if (x > 0 .and. x < 10) then
    print *, "Between 0 and 10"
end if

    Conditional Expressions (Fortran 2023)¶

    Fortran 2023 introduces a ternary-style conditional expression:

    result = (condition ? value_if_true : value_if_false)

    Example:

    value = (a > 0.0 ? a : 0.0)

    Equivalent to:

    if (a > 0.0) then
    value = a
else
    value = 0.0
end if

    Conditional expressions can be nested:

    ivalue = (a < 0.0 ? 0 :
          a < 1.0 ? 1 :
          a < 2.0 ? 2 :
          100)

    Notes:

    • Useful for simple assignments.
    • Often less readable than IF blocks when conditions become complex.
    • All branches must have the same type, kind, and rank.
    • New feature in Fortran 2023; uncommon in existing scientific code.

    Select Case¶

    SELECT CASE is used when choosing between many possible values of the same variable. It is often clearer than a long IF ... ELSE IF chain.

    General form:

    select case (variable)

case (value1)
    ...

case (value2)
    ...

case default
    ...

end select

    Examples:

    Examples:

    select case (age)

    case (:12)
        print *, "Child"

    case (13:17)
        print *, "Teenager"

    case (18:64)
        print *, "Adult"

    case (65:)
        print *, "Senior"

    end select

    Only one CASE block executes. CASE DEFAULT acts like ELSE.

    Works with integers, characters, and other scalar types.

    Use SELECT CASE when asking: "What value does this variable have?"

    Use IF when asking: "Is this condition true?"
