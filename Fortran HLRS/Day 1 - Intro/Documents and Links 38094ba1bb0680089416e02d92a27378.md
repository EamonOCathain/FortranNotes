# Compilers

A **Fortran compiler** is a program that translates Fortran source code into machine code that your computer’s processor can execute.

### **The basic workflow**

You write a file like:

```python
program hello
    print *, "Hello, world!"
end program hello
```

save it as 

```python
hello.f90
```

compile it:

```python
gfortran hello.f90 -o hello
```

The compiler:

1. Reads the Fortran code.
2. Checks for syntax errors.
3. Optimizes the code.
4. Converts it into machine instructions for your CPU.
5. Produces an executable (`hello`).

Then you run:

```python
./hello
```
