# Logical Data

## **Logical Variables**

Logical variables store Boolean values:

```fortran
logical :: flag

flag = .true.
flag = .false.
```

Possible values:

```fortran
.true.
.false.
```

---

### **Using Logical Variables**

Logical variables are commonly used in conditions:

```fortran
logical :: finished

finished = .false.

if (.not. finished) then
    print *, "Still running"
end if
```

---

### **Logical Operators**

| **Operator** | **Meaning** |
| --- | --- |
| `.and.` | AND |
| `.or.` | OR |
| `.not.` | NOT |
| `.eqv.` | Equivalent |
| `.neqv.` | Not equivalent |

Example:

```fortran
if (a > 0 .and. b > 0) then
    print *, "Both positive"
end if
```

---

### **Notes**

- Fortran uses `.true.` and `.false.` instead of `true` and `false`.
- Logical variables are primarily used for conditions, flags, and program flow control.
- Logical values can be printed and are typically displayed as `T` or `F`.
